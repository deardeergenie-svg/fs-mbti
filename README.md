# FS‑MBTI (식품안전 MBTI) – Vite + React

## 빠른 실행
```bash
npm i
npm run dev
```
브라우저: http://localhost:5173

## 빌드
```bash
npm run build
npm run preview
```

## 배포 팁
- Vercel: 새 프로젝트 → Import → Framework: Vite → Build Command: `npm run build`, Output: `dist`
- Netlify: `npm run build` 후 `dist/`를 드래그&드롭 또는 Git 연동
- QR은 배포 URL을 기준으로 자동 생성됩니다.
